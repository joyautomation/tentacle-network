# tentacle-network

Linux network interface monitoring service for the [Tentacle IIoT Platform](https://github.com/joyautomation/tentacle-docs). Publishes network state to NATS for the web dashboard.

## Documentation

See the [Tentacle Docs](https://github.com/joyautomation/tentacle-docs) for:

- [Getting Started](https://github.com/joyautomation/tentacle-docs/blob/main/getting-started.md)
- [Service Documentation](https://github.com/joyautomation/tentacle-docs/tree/main/services)
- [Troubleshooting](https://github.com/joyautomation/tentacle-docs/blob/main/troubleshooting.md)

deno task dev
